## Simple FastAPI Project

This is a simple FastAPI project that provides random motivational quotes, empowering women quotes, funny jokes, and business ideas through API endpoints.

# Features
- **Motivational Quotes**: Get a random motivational quote.

- **Funny Jokes**: Get a random funny joke.

- **Business Ideas**: Get a random business idea.
